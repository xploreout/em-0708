# ACBCC 英文事工 — 使用者指南

本指南说明事工平台中三种登录角色、各角色可执行的功能，以及密码管理方式。

![登录页面](login.jpg)

---

## 登录角色概览

![角色登录概览](rolelogin.jpg)

系统提供三种登录角色，每种角色负责不同的事工管理工作：

| 角色               | 用途                             |
| ------------------ | -------------------------------- |
| **Class（班级）**  | 班级负责人管理出席记录及班级资源 |
| **Coworker（同工）** | 事工同工查看每月服事排班表     |
| **Admin（管理员）** | 系统全面管理                    |

每个角色都有独立密码。在登录页面，请从下拉菜单选择对应角色，并输入该角色密码。

> 若连续 40 分钟没有操作，系统将自动登出并返回登录页面，届时需重新登录。

---

## Class（班级）角色

适用于管理查经班或教育课程的**班长及教师**。

### 班级角色可执行功能

- **查看及管理班级** — 只能访问指派给您的有效班级
- **出席管理** — 为每次聚会签到并记录出席情况
- **管理课程聚会** — 建立聚会记录、记录主题内容，以及添加聚会笔记
- **维护班级成员名单** — 新增成员并查看其出席历史记录
- **查询出席记录** — 跨班级查询个人出席情况
- **上传班级资源** — 上传文件至班级或特定聚会，所有人均可通过菜单列中的 Ministries → Resources 查看这些文件
- **接收电子邮件通知** — 若有未识别人员签到，班长将收到电子邮件通知

### 班长密码

每个班级可由管理员设置独立的**班长密码（Lead Password）**，此密码与 Class 角色登录密码不同。

## Coworker（同工）角色

适用于需要查看完整行事历的**事工同工**，同工也可新增行事历活动。

### 同工角色可执行功能

- **查看服事排班** — 查看月历、周历及日历安排
- **搜索排班** — 查询特定日期或任何时间段内的活动
- **查看教会文件库** — 浏览管理员上传的公告、资源及参考资料

### 同工角色不可执行功能

- 同工无法编辑、新增或删除排班活动 — 仅管理员可执行
- 同工无法上传文件至文件库，仅有查看权限

---

## Admin（管理员）角色

管理员拥有系统所有功能的完整权限。

### 管理员可执行功能

管理员拥有同工角色的所有权限，并额外拥有以下功能：

#### 排班管理

- **新增及编辑活动** — 建立新的服事安排、编辑现有活动，或从行事历中删除活动
- **活动类型管理** — 删除活动类型
- **团队管理** — 删除团队

#### 同工服事提醒邮件

- **发送每月提醒邮件** — 选择任何月份，自动向当月有服事安排的同工发送个别提醒邮件
- 每封邮件将以同工姓名称呼，并列出其所有服事安排，包含日期、活动名称及服事内容
- 系统会回报已成功发送的邮件、被略过的同工（联系人资料不存在或无电子邮件），以及任何发送错误信息

#### 班级管理

- **新增班级** — 建立班级时填写名称、班长资料、聚会地点、聚会时间及重复规则等信息
- **设置班长密码** — 指定或更新班级的班长密码
- **归档班级** — 将不再活跃的班级归档（保留所有历史记录，不会删除资料）

#### 教会文件库管理

- **上传文件** — 上传 PDF、Word、PowerPoint、Excel、图片、视频及音频文件（单个文件最大 100 MB）至共享文件库，所有同工均可查看
- **添加外部链接** — 新增 YouTube 链接或外部网址作为文件库条目
- **编辑文件信息** — 更新任何文件的标题、分类或负责人名称
- **删除文件** — 从文件库中移除文件

#### 会众／联系人管理

- **新增及编辑联系人** — 维护会友通讯录，包含照片资料
- **删除联系人** — 从通讯录中移除成员

#### 密码管理

- **修改角色密码** — 随时更新 Class、Coworker 或 Admin 的登录密码

## 密码管理

### 角色密码（Role Password）

系统共有三组角色密码 — Class、Coworker 及 Admin，同一角色共用同一个密码。

- 仅**管理员（Admin）**可以修改角色密码
- 建议定期更新密码（例如每数个月一次）以维护系统安全
- 密码修改后，请管理员自行通知相关人员 — 系统不会自动发送密码变更通知

### 班长密码（Class Lead Password）

每个班级可拥有独立的班长密码，此密码与 Class 角色登录密码不同，由管理员设定后提供给班长使用。

- 仅**管理员（Admin）**可以设置或重置班长密码
- 若班长需要重置密码，应通知管理员处理
- 班长密码可随时更新，也可定期更换

### 安全建议

- 定期更换角色密码 — 至少在同工或班长离开事工时立即更换
- 请勿将管理员密码分享给非管理员人员
- 管理员密码拥有系统所有资料及设置的完整访问权限，应妥善保管
